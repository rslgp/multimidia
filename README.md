#Projeto de Multimidia
Programa para gerar um video reproduzindo varios videos simultaneos em certa moldura, usando ffmpeg

##Tecnologias usadas:
ffmpeg<br>
java

##Input
- >= 2 videos<br>
- configuração de como os videos input estarão localizados no video output
