Use the drawtext video filter in ffmpeg:

ffmpeg -i input.mp4 -vf drawtext="fontfile=/path/to/font.ttf: \
text='Stack Overflow': fontcolor=white: fontsize=24: box=1: boxcolor=black@0.5: \
boxborderw=5: x=(w-text_w)/2: y=(h-text_h)/2" -codec:a copy output.mp4

  • The @0.5 controls the opacity of the text box. In this example it is set to 50%. You can remove @0.5 and there will be no transparency.

  • See the drawtext filter documentation for more options and examples.

  • -codec:a copy will stream copy (re-mux) the audio and avoid re-encoding.

  • An alternative to the drawtext filter is to use ASS or SRT subtitles–especially if you want timed text.

  • ffmpeg development is very active, so it is recommended to use a recent version if possible. See the FFmpeg Download page for links for Linux, OS X, and Windows builds, or you can always follow a step-by-step compile guide for complete customization.
